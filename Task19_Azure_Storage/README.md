## Objective
Create an Azure Storage Account and explore Blob, File, and Table storage services.

## Steps
Create Storage Account

Open Azure Portal → Storage accounts → Create
Select resource group
Provide storage account name
Region: any region
Performance: Standard
Redundancy: LRS
Create storage account
Blob Storage

Open storage account → Data storage → Containers
Create container (name: mycontainer)
Upload a sample file
File Storage

Open storage account → Data storage → File shares
Create file share (name: myshare)
Upload a file
Table Storage

Open storage account → Data storage → Tables
Create table (name: studenttable)
Verify

Confirm files exist in container and file share
Confirm table created

## Result
The storage account is created and Blob, File, and Table storage services are successfully configured and accessible.
