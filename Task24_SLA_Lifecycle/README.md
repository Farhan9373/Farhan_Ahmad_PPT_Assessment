# Task 24: SLA, Service Lifecycle & Subscription Management

## Simple Definition
Explored Azure SLA, service lifecycle stages, and subscription management.

## Objective
Understand governance and service planning in Azure.

---

## Concepts Covered

### 1. SLA (Service Level Agreement)
SLA defines uptime guarantee provided by Microsoft.
Example: Azure Virtual Machines provide 99.9% uptime SLA.

If SLA is violated, Microsoft provides service credits.

---

### 2. Service Lifecycle
Azure services go through stages:

- Preview (Testing, No SLA)
- General Availability (Production Ready)
- Deprecated/Retired

Preview services do not have SLA.

---

### 3. Subscription Management
Subscription is a billing and resource container in Azure.

Managed through:
Azure Portal → Subscriptions → Cost Management → IAM
