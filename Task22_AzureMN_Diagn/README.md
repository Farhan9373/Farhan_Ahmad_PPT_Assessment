# Task 22: Azure Monitoring & Diagnostics

## Simple Definition
Enabled monitoring and diagnostics for Azure resources using Azure Monitor.

## Objective
Learn diagnostics and performance tracking in Azure.

---

## Services Used

- Azure Monitor
- Virtual Machine
- Log Analytics Workspace
- Diagnostic Settings

---

## Implementation Steps

### 1. Viewed Metrics
- Opened Azure Monitor
- Selected Virtual Machine
- Viewed CPU Percentage metric
- Observed performance graph

### 2. Enabled Diagnostic Settings
- Opened VM → Diagnostic Settings
- Added diagnostic setting
- Sent logs to Log Analytics Workspace
- Saved configuration

---

## Concepts Learned

- Azure Monitor tracks resource performance
- Metrics show real-time performance
- Diagnostic logs store activity data
- Log Analytics helps analyze logs
