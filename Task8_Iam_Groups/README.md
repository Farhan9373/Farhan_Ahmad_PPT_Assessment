## Task 8 – IAM Users and Groups

## Objective
To create IAM users and groups in AWS and manage permissions using group-based policies.

Services Used

## AWS Identity and Access Management (IAM)
Steps Performed
Opened IAM service from the AWS Management Console.
Created an IAM group named Task8-Developers-Group.
Attached the AmazonS3ReadOnlyAccess policy to the group.
Created an IAM user named task8-user.
Enabled AWS Management Console access for the user.
Added the user to the IAM group.
Saved user login credentials.
Verified successful login using the IAM user credentials.

## Outcome
IAM users and groups were successfully created, and permissions were managed using group-based policies.
